# pokemon(demo)

A small, original, Pokémon-inspired demo game — **all original names and assets**. Use it for learning, prototyping, or building a repository to fork.

## What it includes
- Simple overworld "Explore" button that triggers random encounters.
- Turn-based battle with moves, HP, capture mechanic.
- Party management and simple save/load via `localStorage`.
- No external libraries — works by opening `index.html`.

## How to run
1. Clone or download this repo.
2. Open `index.html` in your web browser (Chrome/Firefox recommended).
3. Click **Explore** to encounter wild critters.
